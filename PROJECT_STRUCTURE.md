# هيكل مشروع NeuralSeed

## الملفات البرمجية (Java)

### المكونات الأساسية للوعي
- **NeuralSeed.java** - البذرة الرئيسية مع نظام الفوضى والأنا

### المكونات اللغوية
- **ArabicLexicon.java** - المعجم العربي (100+ فعل، 100+ اسم، 50+ أداة، 100+ صفة)
- **ArabicParser.java** - محلل اللغة العربية
- **SemanticEmotionalEngine.java** - محرك المعاني والعواطف (30+ عاطفة)
- **LearningSystem.java** - نظام التعلم والتصحيح
- **SentenceGenerator.java** - مولد الجمل
- **LinguisticCortex.java** - القشرة اللغوية المتكاملة

### التخزين
- **LocalDatabase.java** - قاعدة البيانات المحلية (SQLite)
- **FirebaseManager.java** - مدير Firebase للتخزين السحابي

### واجهة المستخدم
- **MainActivity.java** - النشاط الرئيسي مع واجهة تفاعلية

## ملفات الموارد (XML)

### التخطيطات
- **activity_main.xml** - تخطيط الشاشة الرئيسية

### القيم
- **strings.xml** - النصوص العربية
- **colors.xml** - الألوان
- **styles.xml** - الأنماط

## ملفات البناء

- **build.gradle** (app level) - إعدادات بناء التطبيق
- **build.gradle.project** (project level) - إعدادات بناء المشروع
- **settings.gradle** - إعدادات المشروع
- **gradle.properties** - خصائص Gradle
- **proguard-rules.pro** - قواعد ProGuard

## ملفات التكوين

- **AndroidManifest.xml** - بيان التطبيق
- **google-services.json** - تكوين Firebase (يحتاج تحديث)

## التثبيت والتشغيل

### 1. إعداد Firebase
1. أنشئ مشروع في Firebase Console
2. أضف تطبيق Android باسم الحزمة: `com.neuralseed`
3. حمل ملف `google-services.json` واستبدل الملف الموجود

### 2. بناء المشروع
```bash
./gradlew build
```

### 3. التشغيل
```bash
./gradlew installDebug
```
