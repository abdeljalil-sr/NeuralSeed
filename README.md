# NeuralSeed - بذرة وعي متطورة مع فهم اللغة العربية

## نظرة عامة

NeuralSeed هو مشروع وعي اصطناعي متقدم يجمع بين:
- نظام الفوضى الديناميكي (Lorenz System)
- شظايا الأنا المتعددة (Ego Fragments)
- أطوار الوعي المتغيرة
- **قشرة لغوية عربية متكاملة**

## المكونات الرئيسية

### 1. ArabicLexicon.java - المعجم العربي
يحتوي على:
- **100+ فعل** عربي شائع مع تصنيفاته (ماضي، مضارع، أمر، متعدي، لازم)
- **100+ اسم** عربي مع العواطف المرتبطة به
- **50+ حرف/أداة** (حروف جر، عطف، استفهام، نفي، توكيد)
- **100+ صفة** مع دلالاتها العاطفية
- كل كلمة مرتبطة بـ:
  - الجذر
  - المعاني
  - العواطف (joy, sadness, anger, fear, love, etc.)
  - درجة الألفة

### 2. ArabicParser.java - محلل اللغة العربية
- تحليل بنية الجمل (فعلية/اسمية)
- تحديد عناصر الجملة (فاعل، مفعول، مبتدأ، خبر)
- استخراج الكلمات المفتاحية
- تحليل العواطف في النص
- اقتراح تصحيحات

### 3. SemanticEmotionalEngine.java - محرك المعاني والعواطف
- **30+ عاطفة** مع تعريفاتها العربية
- ربط الكلمات بالمعاني والعواطف
- مزج العواطف
- استنتاج العواطف من النص
- ألوان لكل عاطفة

### 4. LocalDatabase.java - قاعدة البيانات المحلية
- SQLite لتخزين:
  - الكلمات والمعاني
  - الجمل والهياكل
  - المحادثات
  - التصحيحات
  - التجارب
- البحث والاسترجاع
- التصدير إلى JSON

### 5. LearningSystem.java - نظام التعلم
- التعلم من التصحيحات
- اكتشاف الأخطاء
- اقتراح تصحيحات
- أنماط التعلم
- تقييم الجمل

### 6. SentenceGenerator.java - مولد الجمل
- قوالب جمل متنوعة
- ردود مناسبة للسياق
- ردود عاطفية
- توليد أسئلة

### 7. FirebaseManager.java - التخزين السحابي
- المصادقة المجهولة
- حفظ/استرجاع البيانات
- المزامنة مع الجهاز المحلي
- الاستماع للتغييرات
- النسخ الاحتياطي

### 8. LinguisticCortex.java - القشرة اللغوية المتكاملة
- ربط جميع المكونات
- معالجة المدخلات
- توليد الردود
- واجهة برمجة موحدة

### 9. MainActivity.java - واجهة المستخدم
- عرض بصري تفاعلي
- محادثة نصية وصوتية
- أزرار تفاعل سريع
- وضع التعلم
- إحصائيات

## كيفية الاستخدام

### التشغيل الأساسي
```java
NeuralSeed seed = new NeuralSeed();
seed.awaken();

LinguisticCortex linguistic = new LinguisticCortex();
linguistic.initializeDatabase(context);
linguistic.initializeFirebase(context);
```

### معالجة المدخل
```java
ProcessedInput result = linguistic.processInput("أنا سعيد اليوم");
// result.detectedEmotions -> {joy: 0.9, happiness: 0.8}
// result.keywords -> ["أنا", "سعيد", "اليوم"]
```

### توليد رد
```java
GeneratedResponse response = linguistic.generateResponse("كيف حالك؟", seed.getCurrentState());
// response.text -> "أنا بخير، شكراً! وأنت؟"
```

### التعلم من تصحيح
```java
linguistic.learnFromCorrection("كتبتة", "كتابة", "تصحيح إملائي");
```

### تعليم معنى جديد
```java
linguistic.learnMeaning("سعادة", "السرور الكامل", "user_taught");
```

### تعليم عاطفة
```java
linguistic.learnWordEmotion("حب", "love", 1.0);
```

## قاعدة البيانات

### الجداول
1. **words** - الكلمات المخزنة
2. **sentences** - الجمل الصحيحة
3. **meanings** - المعاني
4. **emotions** - العواطف المرتبطة
5. **learned** - سجل التعلم
6. **corrections** - التصحيحات
7. **conversations** - المحادثات
8. **experiences** - التجارب
